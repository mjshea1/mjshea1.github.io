# Daily UI Challenge

Client: Passion Project
Date: June 2018 - Present
Programs: Sketch, Adobe Photoshop, InVision
Duration: 1 Design per Day

Description:
The Daily UI Design Challenge is a self-directed exercise where you design one of 100 unique user interface pages and elements. The project invites the participant to create a range of UI elements, including an app icon, date picker, and even a currency converter. It was created to provide a fun and engaging way to improve the skills of a designer and to solicit feedback from peers. As a designer, it is my goal to finish the challenge during my career, these are my entries so far.

Challege: Daily UI, Challengers

# Top 5 Designs

# 002 - Checkout

Objective: Design a credit card checkout form or page. Don't forget the important elements such as the numbers, dates, security numbers, etc.




# 007 - Settings

Objective: Design settings for something. Is it for security or privacy settings? Game settings? What is it and what's important?




# 006 - User Profile

Objective: Design a user profile and be mindful of the most important data, names, imagery, placement, etc. Is it for a serious profile? A social profile?




# 008 - 404 Page

Objective: Design a 404 page. Does it suit the brand's style? Is it user-friendly?




# 004 - Calculator

Objective: Design a calculator. Standard, scientific, or specialty calculator for something such as a mortgage? Is it for a phone, a tablet, a web app?
