# My Portfolio Journey

An online portfolio is a designers intimate and personal expression of their artistic work and skillset. The journey of my portfolio has taken many forms since its first iteration in 2004. Through its progression it stands as a snapshot of my relationship to the time's technology combined with design trends. My first portfolio was entirely in flash, a very popular web solution at the time. Today I work in VuePress, a vue-powered static site generator, to display my design experience and document my continuing research in UI Design and design as a whole. Taking a detailed approach to present design work, I aim to show experience and technique while providing insight into my design process.

# brenicor

Version 1: Portfolio in Flash
Date: 2004 - 2008

My first online portfolio was deployed over 15 years ago. During this time, Macromedia Flash was all the rage. Many popular websites were created with long complicated introductions, flashy motion and music to accompany the intro. With this being the trend, I built my entire site in Flash with each project as a unique pop up window. My portfolio even featured three unique songs created by my talented colleague, DJ Eric Ryan. And if you are wondering, yes, that is the papyrus font.


# James Shea Design

Version 2: Wordpress Portfolio
Date: 2009 - 2012

For my second iteration of my portfolio, I decided to use WordPress to showcase my work. WordPress is a content management system website builder, and is often used to create sites that are content heavy. CMS's were, and still are, a popular solution. I incorporated many new features, including an image slider, a robust contact page, and a blog. My goal with this portfolio was to have the design be less intrusive and illustrate more design projects.


# JSD

Version 3: HTML Template
Date: 2012 - 2015

The inspiration for my third portfolio was a new design trend around simplifying and focusing the user experience. It was to combine all the individual pages and functions of a website onto one page. One Page Love features many successful examples of this movement that still exist today, often seen in independent App start ups. To construct this iteration I purchased a basic HTML template and personalized the hell out of the design. It also contained new features; the ability to filter or sort projects, and an animated scrolling effect with a sticky header.


# JSD 2.0

Version 4: Custom Responsive Design
Date: 2015 - 2019

Beginning around 2012 people were increasing to accessing the web through smartphones and tablets. With Google Analytics installed on my portfolio I could see I was not immune to the new trend. I knew I needed a redesign that was fully responsive and supportive of new users. Also, with the growing popularity in flat design, and a content-first approach, it was apparent a refresh was necessary. With my expanding knowledge of web development, I decided to create a site entirely from scratch. This second version of JSD featured a unique parallax-scrolling header for each page, animated portfolio sorting, and a fun direction-aware hover effect.


# MSD

Version 5: Static Site Generator
Date: 2019 - Present

In the latest version of my portfolio I intended to change focus. Instead of a heavy focus on design projects, I built a site that was home to my continuing research in User Interface Design. I was inspired by my research into atomic design as a foundation of Design Systems and the documentation that was needed for guidance. I explored many unique solutions to solve my new direction including online templates, wiki based websites, and document site builders. My first version was built from scratch like I had done previously, and then I discovered static site generators, like VuePress. So I learned new web development skills to be able to create a portfolio with my new direction.
