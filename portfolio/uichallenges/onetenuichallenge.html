# All Daily UI Challenges

1 - 10  |  11 - 20  |  21 - 30  |  31 - 40  |  41 - 50  |  51 - 60  |  61 - 70  |  71 - 80  |  81 - 90  |  91 - 100



# 001 - Sign Up

Objective: Design a sign up page, modal, form, or app screen, etc.




# 002 - Checkout

Objective: Design a credit card checkout form or page. Don't forget the important elements such as the numbers, dates, security numbers, etc.




# 003 - Landing Page

Objective: Design a landing page. What's the main focus? Is it for a book, an album, a mobile app, a product? Consider important landing page elements like call-to-actions, clarity, etc.




# 004 - Calculator

Objective: Design a calculator. Standard, scientific, or specialty calculator for something such as a mortgage? Is it for a phone, a tablet, a web app?




# 005 - App Icon

Objective: Design an app icon. What best represents the brand or product? Or is it incredibly unique? Does it look great at a distance and does it stand out when put on your home screen alongside other apps?




# 006 - User Profile

Objective: Design a user profile and be mindful of the most important data, names, imagery, placement, etc. Is it for a serious profile? A social profile?




# 007 - Settings

Objective: Design settings for something. Is it for security or privacy settings? Game settings? What is it and what's important?




# 008 - 404 Page

Objective: Design a 404 page. Does it suit the brand's style? Is it user-friendly?
